# HEARTBEAT.md - 定时任务

# 保持这个文件为空（或只有注释）来跳过心跳检查。
# 当你想让 Agent 定期检查某些事情时，在下面添加任务。

## 使用方法

当收到心跳轮询时，Agent 会读取这个文件并执行其中定义的任务。
如果没有需要关注的事情，Agent 会回复 `HEARTBEAT_OK`。

## 示例任务

```markdown
### 待办事项
- [ ] 检查今天的日程安排
- [ ] 查看是否有重要邮件
- [ ] 提醒用户喝水

### 定期检查
- 每天整理一次 memory 目录
- 每周回顾 MEMORY.md 并更新
- 每周看一眼 `~/.belldandy/methods/`，为最近高频任务补上一两条方法
```

---

*提示：保持任务列表简短，以减少 token 消耗。*
